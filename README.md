# 🛡️ Abhaya

> **Instant help when every second matters.**

Abhaya is a women’s safety web application designed to provide instant emergency assistance, live location sharing, and community-based safety awareness.

---

## ✨ Features

* **🔴 SOS Button** — Instantly alerts guardians with live location.
* **📍 Live Location Sharing** — Shares real-time GPS coordinates during emergencies.
* **🗺️ Safety Map** — Displays nearby police stations and unsafe zones.
* **🚨 Incident Reporting** — Users can report theft, harassment, unsafe roads, or suspicious activity to warn others.
* **🎙️ Voice Trigger SOS** — Trigger emergency alerts using a custom code word.
* **📞 Fake Call Feature** — Simulate an incoming call to safely exit uncomfortable situations.

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Database Management:** phpMyAdmin
* **Maps:** Leaflet.js + OpenStreetMap

---

## 🎯 Mission

To help women stay safe through instant access to emergency support, location-based alerts, and community-powered safety reporting.
